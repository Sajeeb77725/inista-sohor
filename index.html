How does JS work ?

ANS: JavaScript is a high-level, single-threaded and scripted programming language. Because of single threaded type javaScripte has a single call stack. Call Stack is a data structure which records something like where we doing program. Because of single threaded type it can handle one task at a time. Also It is an interpreted programming language and it uses dynamic typing. Where we can check type safety at the runtime. It is a high-level language because it does'n require direct interaction with the operating system and hardware.

What does event loop do ?

ANS: JavaScript has a runtime model based on an event loop and it is responsible for executing the code. After that collecting, processing events and executing queued sub-tasks. JavaScript is single-threaded, but the browser APIs act as separate threads. The event loop make this process easy and it constantly checks whether or not the call stack is empty. If the call stack is empty then new functions are added from the event queue. If it's not empty, then the current function call is processed.
